# 《三角洲行动》烽火地带游戏系统分析

本仓库整理了《三角洲行动》核心模式「烽火地带」的全维度系统分析，涵盖玩法、干员、经济、养成、地图等模块，并针对现存问题提出优化建议。

## 核心分析维度
1. **核心玩法系统**：PVPVE 闭环设计、曼德尔砖机制、AI/BOSS 体系
2. **干员战术系统**：四大兵种定位、小队协作、平衡问题
3. **物资经济系统**：货币体系、库存管理、通胀痛点
4. **角色养成系统**：轻量化养成、等级/任务体系
5. **地图场景系统**：资源分布、动态机制
6. **优化展望**：系统平衡、地图迭代、经济通胀解决方案

## 优化重点
- 经济通胀：调控战备成本、拓宽哈夫币消耗渠道、重塑高阶物资价值
- 干员平衡：弱化强势干员泛用性、丰富战术多样性
- 地图迭代：动态调整资源分布、激活热点区域博弈
